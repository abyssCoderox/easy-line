## 1. 数据存储概述

### 1.1 Demo版存储策略

Demo版本采用精简的存储策略，避免引入数据库依赖：

| 数据类型 | 存储方式 | 说明 |
|---------|---------|------|
| 对话上下文 | 内存 (Map) | 使用 LangChain BufferMemory |
| 定时任务配置 | JSON文件 | `config/tasks.json` |
| 环境配置 | 环境变量 | `.env` 文件 |

### 1.2 为什么不用数据库

| 原因 | 说明 |
|------|------|
| 快速开发 | 避免数据库安装、配置、迁移 |
| Demo验证 | 仅需验证核心功能，无需持久化 |
| 简化部署 | 减少依赖，降低部署复杂度 |
| 成本控制 | 无需额外数据库服务器 |

### 1.3 后续扩展

正式版本可扩展为 PostgreSQL + Redis 存储：
- PostgreSQL: 持久化用户、对话、任务数据
- Redis: 缓存对话上下文、任务锁

## 2. 内存数据结构

### 2.1 对话上下文存储

使用 LangChain 的 `BufferMemory` 管理对话上下文。

**TypeScript 类型定义：**

```typescript
import { BufferMemory } from 'langchain/memory';

export class LLMService {
  private memories: Map<string, BufferMemory> = new Map();
  
  private getMemory(userId: string): BufferMemory {
    if (!this.memories.has(userId)) {
      this.memories.set(userId, new BufferMemory({
        returnMessages: true,
        memoryKey: 'history'
      }));
    }
    return this.memories.get(userId)!;
  }
}
```

**内存结构：**

```
Map<string, BufferMemory>
├── "U1234567890" → BufferMemory
│   └── history: [
│       { role: "user", content: "你好" },
│       { role: "assistant", content: "你好！有什么可以帮助你的吗？" }
│   ]
├── "U0987654321" → BufferMemory
│   └── history: [...]
└── ...
```

### 2.2 定时任务内存存储

使用 `node-cron` 的 `ScheduledTask` 对象管理任务。

**TypeScript 类型定义：**

```typescript
import cron from 'node-cron';

export class SchedulerService {
  private tasks: Map<string, cron.ScheduledTask> = new Map();
}
```

**内存结构：**

```
Map<string, ScheduledTask>
├── "weather-push" → ScheduledTask (cron job)
├── "news-push" → ScheduledTask (cron job)
└── ...
```

## 3. JSON配置文件设计

### 3.1 定时任务配置文件

**文件路径：** `config/tasks.json`

**数据结构：**

```typescript
interface TaskConfig {
  id: string;
  name: string;
  enabled: boolean;
  schedule: string;
  api: {
    url: string;
    method: 'GET' | 'POST';
    headers?: Record<string, string>;
    body?: Record<string, any>;
  };
  template: string;
  targets: string[];
}
```

**完整示例：**

```json
{
  "tasks": [
    {
      "id": "weather-push",
      "name": "每日天气推送",
      "enabled": true,
      "schedule": "0 8 * * *",
      "api": {
        "url": "https://api.openweathermap.org/data/2.5/weather",
        "method": "GET",
        "headers": {
          "Accept": "application/json"
        }
      },
      "template": "早安！今日天气: {weather}, 温度: {temp}°C, 湿度: {humidity}%",
      "targets": ["U1234567890abcdef"]
    },
    {
      "id": "news-push",
      "name": "每日新闻推送",
      "enabled": true,
      "schedule": "0 9 * * *",
      "api": {
        "url": "https://news-api.example.com/v1/top-headlines",
        "method": "GET",
        "headers": {
          "Authorization": "Bearer ${NEWS_API_KEY}"
        }
      },
      "template": "今日头条:\n{headlines}",
      "targets": ["U1234567890abcdef", "U0987654321fedcba"]
    }
  ]
}
```

### 3.2 字段说明

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | string | 是 | 任务唯一标识 |
| name | string | 是 | 任务名称（用于日志） |
| enabled | boolean | 是 | 是否启用 |
| schedule | string | 是 | Cron表达式 |
| api.url | string | 是 | 第三方API地址 |
| api.method | string | 是 | HTTP方法 (GET/POST) |
| api.headers | object | 否 | 请求头（支持环境变量占位符） |
| api.body | object | 否 | 请求体（POST时使用） |
| template | string | 是 | 消息模板（支持变量占位符） |
| targets | string[] | 是 | 目标用户ID列表 |

### 3.3 模板变量

模板中使用 `{variable}` 格式引用API响应数据：

```typescript
private renderTemplate(template: string, data: Record<string, any>): string {
  return template.replace(/\{(\w+)\}/g, (_, key) => {
    return data[key] !== undefined ? String(data[key]) : '';
  });
}
```

**示例：**

- 模板: `"今日天气: {weather}, 温度: {temp}°C"`
- 数据: `{ weather: "晴", temp: 25 }`
- 结果: `"今日天气: 晴, 温度: 25°C"`

### 6.4 LLM服务

```typescript
// src/services/llm.ts
import { ChatOpenAI } from '@langchain/openai';
import { BufferMemory } from 'langchain/memory';
import { ConversationChain } from 'langchain/chains';
import { config } from '../config';

export class LLMService {
  private model: ChatOpenAI;
  private memories: Map<string, BufferMemory> = new Map();

  constructor() {
    this.model = new ChatOpenAI({
      modelName: 'gpt-3.5-turbo',
      temperature: 0.7,
      openAIApiKey: config.openai.apiKey
    });
  }

  private getMemory(userId: string): BufferMemory {
    if (!this.memories.has(userId)) {
      this.memories.set(userId, new BufferMemory({
        returnMessages: true,
        memoryKey: 'history'
      }));
    }
    return this.memories.get(userId)!;
  }

  async chat(userId: string, message: string): Promise<string> {
    const memory = this.getMemory(userId);
    
    const chain = new ConversationChain({
      llm: this.model,
      memory: memory
    });

    const response = await chain.call({ input: message });
    return response.response;
  }
}
```

### 6.5 定时任务服务

```typescript
// src/services/scheduler.ts
import cron from 'node-cron';
import { Client } from '@line/bot-sdk';
import axios from 'axios';
import { TaskConfig } from '../types';

export class SchedulerService {
  private tasks: Map<string, cron.ScheduledTask> = new Map();
  
  constructor(private client: Client) {}

  loadTasks(taskConfigs: TaskConfig[]): void {
    for (const task of taskConfigs) {
      if (!task.enabled) {
        console.log(`Task skipped (disabled): ${task.name}`);
        continue;
      }
      
      if (!cron.validate(task.schedule)) {
        console.error(`Invalid cron: ${task.schedule} for task: ${task.name}`);
        continue;
      }
      
      const scheduledTask = cron.schedule(task.schedule, () => {
        this.executeTask(task);
      });
      
      this.tasks.set(task.id, scheduledTask);
      console.log(`Task loaded: ${task.name} (${task.schedule})`);
    }
  }

  private async executeTask(task: TaskConfig): Promise<void> {
    console.log(`Executing task: ${task.name}`);
    const startTime = Date.now();
    
    try {
      const response = await axios({
        method: task.api.method,
        url: task.api.url,
        headers: task.api.headers,
        data: task.api.body
      });

      const message = this.renderTemplate(task.template, response.data);
      
      await this.client.multicast(task.targets, {
        type: 'text',
        text: message
      });
      
      const duration = Date.now() - startTime;
      console.log(`Task completed: ${task.name} (${duration}ms)`);
    } catch (error) {
      console.error(`Task failed: ${task.name}`, error);
    }
  }

  private renderTemplate(template: string, data: Record<string, any>): string {
    return template.replace(/\{(\w+)\}/g, (_, key) => {
      return data[key] !== undefined ? String(data[key]) : '';
    });
  }

  stopTask(taskId: string): boolean {
    const task = this.tasks.get(taskId);
    if (task) {
      task.stop();
      this.tasks.delete(taskId);
      return true;
    }
    return false;
  }

  stopAll(): void {
    for (const [id, task] of this.tasks) {
      task.stop();
    }
    this.tasks.clear();
  }
}
```
